Moved to a Droplet

Sunday, 29 June 2014

Having wished for a while to run this little site on SSL —as well as having the freedom to do more things— which GitHub does not allow, I have finally decided to move to a virtual server of my own. At $5 a month per droplet, DigitalOcean was the provider of choice.

DigitalOcean droplets web interface is very clean, simple, responsive and functional. Spawning the server took 54 seconds, and configuring nginx took 15 minutes, SSL included. I ordered and received my SSL certificate while quickly configuring nginx; kudos to Namecheap and Comodo for a speedy service.

1992 - 2020 — David Collantes.
